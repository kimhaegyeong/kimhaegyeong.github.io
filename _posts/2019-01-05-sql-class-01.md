### 테이블 생성
```sql
CREATE TABLE `membertbl` (
  `memerID` varchar(15) NOT NULL,
  `memberName` varchar(10) NOT NULL,
  `memberAddress` varchar(45) NOT NULL,
  PRIMARY KEY (`memerID`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_0900_ai_ci;
```
